# 🛒 Superstore Sales Analysis — SQL & Power BI

## 📌 Project Overview
This project analyses the Kaggle Superstore dataset to uncover insights on sales performance, profitability, customer behaviour, and operational efficiency using SQL and Power BI.

## 🛠️ Tools Used
- **SQL** (SQLite) — Data extraction and analysis
- **Power BI** — Interactive dashboard and visualizations
- **Dataset** — [Superstore Dataset, Kaggle](https://www.kaggle.com/datasets/vivek468/superstore-dataset-final)

## 📊 Dashboard Pages
| Page | Focus |
|---|---|
| Sales & Revenue | Year-on-year growth, top products, category share |
| Profit & Loss | Regional profit, loss-making products, negative profit states |
| Customer Segmentation | Segment analysis, top customers, repeat buyers |
| Operational Insights | Discount impact on profit, region × category matrix, shipping analysis |

## 🔍 Key Business Insights
- Sales grew consistently from 2014 to 2017, with Technology leading in revenue
- West region has the highest profit margin; Central region is underperforming
- High discounts (40%+) consistently result in losses — discounting strategy needs review
- Tables and Bookcases are loss-making sub-categories despite Furniture having decent sales
- Consumer segment drives the most orders but Corporate yields better profit margins
